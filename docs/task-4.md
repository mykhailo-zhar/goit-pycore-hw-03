# Завдання 4

У межах вашої організації ви відповідаєте за організацію привітань колег з днем народження. Щоб оптимізувати цей процес, вам потрібно створити функцію `get_upcoming_birthdays`, яка допоможе визначати, кого з колег потрібно привітати. Функція повинна повертати список усіх, у кого день народження протягом наступних 7 днів включно з поточним днем.

У вашому розпорядженні є список `users`, де кожен елемент містить інформацію про ім'я користувача і його день народження. Оскільки дні народження можуть припадати на вихідні, функція також має переносити дату привітання на наступний робочий день, якщо потрібно.

---

## Вимоги до завдання

- **Параметр функції `users`** — це список словників, де кожен словник містить ключі `name` (ім'я користувача, рядок) та `birthday` (день народження, рядок у форматі `'рік.місяць.дата'`).
- **Функція визначає**, чиї дні народження випадають у найближчі 7 днів включно з поточним днем. Якщо день народження припадає на вихідний, дата привітання переноситься на найближчий понеділок.
- **Функція повертає** список словників, де кожен словник містить інформацію про користувача (`name`) та дату привітання (`congratulation_date` у форматі рядка `'рік.місяць.дата'`).

---

## Рекомендації для виконання

1. Припускаємо, що ви отримали список `users`, де кожен словник містить `name` (ім'я користувача) та `birthday` (дата народження у форматі рядка `'рік.місяць.дата'`).
2. Перетворіть дати народження з рядка у об'єкти `datetime` за допомогою `datetime.strptime(user["birthday"], "%Y.%m.%d").date()`.
3. Отримайте поточну дату за допомогою `datetime.today().date()`.
4. Пройдіться по списку користувачів та аналізуйте дати народження кожного (`for user in users:`).
5. Перевірте, чи вже минув день народження цього року (`if birthday_this_year < today`). Якщо так, врахуйте наступний рік.
6. Визначте, чи день народження відбудеться протягом наступних семи днів.
7. Перевірте, чи день народження припадає на вихідний. Якщо так, перенесіть дату привітання на наступний понеділок.
8. Створіть структуру даних, яка зберігає ім'я користувача та відповідну дату привітання, якщо день народження відбудеться протягом наступного тижня.
9. Поверніть зібрані дані у вигляді списку словників з іменами користувачів та датами привітань.

---

## Критерії оцінювання

- Актуальність та коректність визначення днів народження на 7 днів вперед.
- Правильність обробки випадків, коли дні народження припадають на вихідні.
- Читабельність і структурованість коду.

---

## Приклад

Припустимо, у вас є список:

```python
users = [
    {"name": "John Doe", "birthday": "1985.01.23"},
    {"name": "Jane Smith", "birthday": "1990.01.27"}
]
```

Використання функції:

```python
upcoming_birthdays = get_upcoming_birthdays(users)
print("Список привітань на цьому тижні:", upcoming_birthdays)
```

Якщо сьогодні 2024.01.22, результат може бути:

```python
[
    {'name': 'John Doe', 'congratulation_date': '2024.01.23'},
    {'name': 'Jane Smith', 'congratulation_date': '2024.01.29'}
]
```

Цей список містить інформацію про те, кого і коли потрібно привітати з днем народження.

## Поведінка функції
1. Функція `get_upcoming_birthdays(users)` приймає масив словників формату `input` та повертає масив словників формату `output`, де `birthdate` та `congratulation_date` є рядками формату `\d{4}\.\d{2}\.\d{2}`. 
1. При невідповідності типу users масиву функція має згенерувати виключення `TypeError`.
1. При відсутності необхідних ключів функція має згенерувати виключення `ValueError`.
1. Функція повертає тих користувачів, день народження яких припадає на наступні 7 днів, включаючи поточний. 
1. Функція переносить вітання користувачів, день народження яких припадає на вихідні, на перший понеділок після вихідних.

## Формати

input: 
```
[
  {
    "name": str,
    "birthday": str
  }
]
```

output:
```
[
  {
    "name": str,
    "congratulation_date": str
  }
]
```
