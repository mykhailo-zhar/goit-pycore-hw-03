# Завдання 1

Створіть функцію `get_days_from_today(date)`, яка розраховує кількість днів між заданою датою та поточною датою.

## Вимоги до завдання:

- Функція приймає один параметр: `date` — рядок, що представляє дату у форматі `'РРРР-ММ-ДД'` (наприклад, `'2020-10-09'`).
- Функція `get_days_from_today(date)` повертає ціле число (`int`), яке вказує на кількість днів від заданої дати до поточної. Якщо значення `date` пізніше поточної дати, результат має бути від'ємним.
- У розрахунках необхідно враховувати лише дні, ігноруючи час (години, хвилини, секунди).
- Для роботи з датами використовуйте модуль `datetime` мови Python.

## Рекомендації для виконання:

1. Імпортуйте модуль` datetime`.
2. Перетворіть рядок дати у форматі `'РРРР-ММ-ДД'` у об'єкт `datetime`.
3. Отримайте поточну дату, використовуючи `datetime.today()`.
4. Розрахуйте різницю між поточною датою та заданою датою.
5. Поверніть різницю у днях як ціле число.

## Критерії оцінювання:

- **Коректність роботи функції**: функція `get_days_from_today(date)` повинна точно обраховувати кількість днів між датами.
- **Обробка винятків**: функція має впоратися з неправильним форматом вхідних даних.
- **Читабельність коду**: код функції має бути чистим і добре документованим.

## Приклад використання:

Якщо сьогодні 5 травня 2021 року, виклик

```python
get_days_from_today("2021-10-09")
```

повинен повернути `-157`, оскільки 9 жовтня 2021 року на 157 днів пізніше від 5 травня 2021 року.

## Поведінка функції

1. Якщо функцію `get_days_from_today(date)` викликають зі строковим параметром `date` у форматі `'РРРР-ММ-ДД'`, тоді функція повертає число типу `int`.
2. Якщо `date` не є рядком, функція повинна згенерувати помилку `TypeError`.
3. Якщо рядок `date` має неправильний формат (не `'РРРР-ММ-ДД'`), функція повинна згенерувати помилку `ValueError`.
4. Якщо передано валідний параметр `date`, функція повертає число, яке вказує на кількість днів `x` від заданої дати до поточної дати (`x < 0` — якщо дата пізніша за поточну, `x = 0` — якщо дата дорівнює поточній, `x > 0` — якщо дата раніше за поточну).
